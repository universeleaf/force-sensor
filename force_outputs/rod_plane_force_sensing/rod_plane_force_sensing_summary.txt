Implementation note:
The estimate does not use the forward solver contact force or contact index. It uses the measured shape and measured plane to infer a contact candidate, then estimates both the contact force and tip load in the reduced constrained MAP state x=[p1; eta1; s1; f1n; beta1; lambda1; fe]. This is not the full recursive iterated EKF/MAP in Formulation.pdf eqs. (19)-(29); it is a shape-known force-balance subproblem using the same state variables and force constraints. Aloi is used only as a shape-only Gaussian total-load baseline and does not receive the plane/friction constraints.
